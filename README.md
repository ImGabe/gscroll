# gscroll

> Marquee for bar and other things 

## Usage

```sh
Usage of ./gscroll:
  -length int
        length of characters to scroll. (default 20)
  -marquee
        Text scroll left. (default true)
  -speed int
        Speed of text scrolling. (default 5)
  -text string
        Text to scroll.
```

## Example

```sh
$ ./gscroll -text "Golang is great"
```
